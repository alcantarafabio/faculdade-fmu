# Técnicas de Machine Learning

Este diretório reúne materiais utilizados durante os estudos da disciplina de **Técnicas de Machine Learning** na **Escola Tech** do **Centro Universitário FMU | FIAM–FAAM**.

O conteúdo está organizado em notebooks e conjuntos de dados utilizados ao longo da disciplina para acompanhar os conceitos apresentados em aula e realizar os exercícios propostos.

## Estrutura

```text
machine-learning-estudos/
├── datasets/
│   ├── alunos_dados.xlsx
│   └── titanic.xlsx
├── notebooks/
│   ├── 01_preprocessamento_dados.ipynb
│   └── 02_algoritmos_machine_learning.ipynb
├── .gitignore
├── README.md
└── requirements.txt
```

## Conteúdo

### datasets

Conjuntos de dados utilizados nos notebooks da disciplina.

### notebooks

- **01_preprocessamento_dados.ipynb** — Introdução ao pré-processamento e preparação de dados.
- **02_algoritmos_machine_learning.ipynb** — Exemplos de aplicação de algoritmos de Machine Learning.

## Requisitos

As dependências necessárias para executar os notebooks estão listadas no arquivo `requirements.txt`.

Para instalá-las, execute:

```bash
pip install -r requirements.txt
```
